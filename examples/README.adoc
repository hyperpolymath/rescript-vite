== Overview

This directory contains practical examples demonstrating common patterns in ReScript + Vite applications. Each example is type-safe and self-contained.

== FormExample.res

**What it shows:** Type-safe form handling with validation.

- Demonstrates form state management with `React.useState`
- Type-safe field updates using variant pattern matching
- Real-time validation feedback
- Error message rendering
- Form submission handling

**Key patterns:**
- Record types for structured form data
- Option types for nullable error messages
- Pattern matching for conditional rendering

**Run it:**
```bash
# Copy into src/rescript/FormExample.res and import into App.res
import FormExample from '@/FormExample'
```

== ApiExample.res

**What it shows:** Type-safe asynchronous data fetching from APIs.

- Demonstrates `Js.Promise` with `async`/`await`
- Variant types for managing async states (Loading, Success, Error)
- Error handling and recovery
- Rendering based on state

**Key patterns:**
- State variants for representing different async states
- `React.useEffect0` for side effects
- Promise-based async operations
- Type-safe error handling

**Run it:**
```bash
# Copy into src/rescript/ApiExample.res and import into App.res
import ApiExample from '@/ApiExample'
```

**Note:** The current example fetches from JSONPlaceholder (public API). In production, you'd:
1. Add a proper JSON decoder (e.g., `@rescript/core` or a custom decoder)
2. Handle authentication (Bearer tokens, API keys)
3. Add retry logic for failed requests
4. Implement caching or request deduplication

## Common Patterns Across Examples

=== Type Safety

All examples leverage ReScript's type system:
- **Records** for structured data
- **Variants** for enums and polymorphic states
- **Options** for nullable values
- **Pattern matching** for conditional logic

=== State Management

All examples use `React.useState` for local state:
```rescript
let (state, setState) = React.useState(_ => initialValue)
```

=== Event Handling

Consistent event handling patterns:
```rescript
onChange={e => handleChange(ReactEvent.Form.target(e)["value"])}
onClick={_ => handleClick()}
onSubmit={e => {
  ReactEvent.Form.preventDefault(e)
  // Handle submission
}}
```

=== Conditional Rendering

Using pattern matching for clean conditionals:
```rescript
{switch state {
| Loading => <p>Loading...</p>
| Success(data) => <div>{render(data)}</div>
| Error(err) => <div>Error: {err}</div>
}}
```
